# OOP-Project
### Smart University Management System

A is a console-based Java application designed to demonstrate all core Object-Oriented Programming concepts. The system simulates a real-world university environment and covers user management, course administration, enrollment, grading, and system-wide operations through a structured, modular architecture.
The project is divided into four interconnected modules, each targeting specific OOP principles while contributing to a cohesive, working system.
